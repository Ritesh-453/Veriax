Code Examples
cURL — Scan an image
curl -X POST http://127.0.0.1:5000/api/v1/scan \
  -H "X-API-Key: ss_yourkeyhere" \
  -F "image=@suspect_logo.png"
Python — Register an asset
import requests

res = requests.post(
  "http://127.0.0.1:5000/api/v1/register",
  headers={"X-API-Key": "ss_yourkeyhere"},
  files={"image": open("logo.png", "rb")},
  data={"name": "IPL Logo 2026"}
)
print(res.json())
